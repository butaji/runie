### Create Transport from TCP Stream (Client)

Source: https://docs.rs/rmcp/latest/rmcp/transport/index.html

This example demonstrates how to create a client transport using a TCP stream. It requires the `client` feature to be enabled. The code connects to a specified address, serves the stream as a client transport, and then lists available tools.

```rust
#[cfg(feature = "client")]

// create transport from tcp stream
#[cfg(feature = "client")]
async fn client() -> Result<(), Box<dyn std::error::Error>> {
    let stream = tokio::net::TcpSocket::new_v4()? 
        .connect("127.0.0.1:8001".parse()?)
        .await?;
    let client = ().serve(stream).await?;
    let tools = client.peer().list_tools(Default::default()).await?;
    println!("{:?}", tools);
    Ok(())
}
```
