### Exponential Backoff Retry Example

Source: https://docs.rs/backon/1.6.0/src/backon/backoff/exponential.rs.html

Demonstrates how to use ExponentialBuilder with the retry method to fetch content from a URL, retrying exponentially on failure. Requires the `reqwest` and `anyhow` crates.

```rust
use anyhow::Result;
use backon::ExponentialBuilder;
use backon::Retryable;

async fn fetch() -> Result<String> {
    Ok(reqwest::get("https://www.rust-lang.org")
        .await?
        .text()
        .await?)
}

#[tokio::main(flavor = "current_thread")]
async fn main() -> Result<()> {
    let content = fetch.retry(ExponentialBuilder::default()).await?;
    println!("fetch succeeded: {}", content);

    Ok(())
}
```

### Example: Fetch with Retry and Notification

Source: https://docs.rs/backon/1.6.0/src/backon/retry.rs.html

Demonstrates how to use `retry` with `ExponentialBuilder` and `notify` to fetch content, retrying on errors and logging retry attempts.

```rust
use core::time::Duration;

use anyhow::Result;
use backon::ExponentialBuilder;
use backon::Retryable;

async fn fetch() -> Result<String> {
    Ok(reqwest::get("https://www.rust-lang.org")
        .await?
        .text()
        .await?)
}

#[tokio::main(flavor = "current_thread")]
async fn main() -> Result<()> {
    let content = fetch
        .retry(ExponentialBuilder::default())
        .notify(|err: &anyhow::Error, dur: Duration| {
            println!("retrying error {:?} with sleeping {:?}", err, dur);
        })
        .await?;
    println!("fetch succeeded: {}", content);

    Ok(())
}
```

### Retry an async function with Exponential Backoff

Source: https://docs.rs/backon/1.6.0/backon/docs/examples/basic/index.html

This snippet shows how to retry an asynchronous function using `ExponentialBuilder` from the backon crate. It requires the `tokio` runtime and `anyhow` for error handling.

```rust
use backon::ExponentialBuilder;
use backon::Retryable;
use anyhow::Result;

async fn fetch() -> Result<String> {
    Ok("Hello, World!".to_string())
}

#[tokio::main]
async fn main() -> Result<()> {
    let content = fetch.retry(ExponentialBuilder::default()).await?;

    println!("fetch succeeded: {}", content);
    Ok(())
}
```
