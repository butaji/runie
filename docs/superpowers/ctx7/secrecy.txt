### ExposeSecret Trait Definition

Source: https://docs.rs/secrecy/latest/src/secrecy/lib.rs.html

Defines the `ExposeSecret` trait, which provides a method to get a read-only reference to the inner secret. This is the primary mechanism for accessing secret data.

```rust
pub trait ExposeSecret<S: ?Sized> {
    /// Expose secret: this is the only method providing access to a secret.
    fn expose_secret(&self) -> &S;
}
```
