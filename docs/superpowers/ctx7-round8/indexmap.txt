### indexmap!

Source: https://docs.rs/indexmap/latest/src/indexmap/macros.rs.html

Creates an IndexMap from a list of key-value pairs.

```APIDOC
## indexmap!

### Description
Creates an IndexMap from a list of key-value pairs.

### Parameters
- **$key => $value** (expr) - Required - Key-value pairs to insert into the map.
```

--------------------------------

### Creating an IndexMap using indexmap! macro

Source: https://docs.rs/indexmap/latest/index.html

This macro creates an `IndexMap` from a list of key-value pairs. It requires the `std` feature to be enabled.

```rust
use indexmap::indexmap;

let map = indexmap! {
    "a" => 1,
    "b" => 2,
};
assert_eq!(map.len(), 2);
```

--------------------------------

### Initialize IndexMap with macro

Source: https://docs.rs/indexmap/latest/indexmap/macro.indexmap.html

Create and populate an IndexMap using the indexmap! macro syntax.

```rust
use indexmap::indexmap;

let map = indexmap!{
    "a" => 1,
    "b" => 2,
};
assert_eq!(map["a"], 1);
assert_eq!(map["b"], 2);
assert_eq!(map.get("c"), None);

// "a" is the first key
assert_eq!(map.keys().next(), Some(&"a"));
```

### IndexMap > Overview

Source: https://docs.rs/indexmap/latest/src/indexmap/map.rs.html

IndexMap is a hash table implementation where the iteration order of key-value pairs is independent of the hash values of the keys. Its interface is closely compatible with the standard HashMap, while offering additional features.
